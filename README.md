# SimplePairing

A very simple pairing code that takes in all members and groups them into a group of two by shuffling randomly.

# Example

```
 ArrayList<String> names=new ArrayList<>();
        names.add("A");
        names.add("B");
        names.add("C");
        names.add("D");
        names.add("E");
        names.add("F");
        names.add("G");
        names.add("H");
        names.add("I");
 ```
# Result
  ```
 Group 1-D,G
 Group 2-C,F
 Group 3-H,B
 Group 4-A,J
 Group 5-E,I
  ```
